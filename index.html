Обзор

Описание

Основные возможности меню

  1. Поддержка бесконечного числа уровней вложенности
  2. Поддержка swipe-раскрытия и swipe-закрытия меню
  3. Адаптивность стилей
  4. Легкая и быстрая настройка

Системные требования

  • Jquery 1.7+
  • Android: 4.2.2+
  • IOS 7.1+

Работа с плагином

Подключение материалов и создание Html-структуры

Шаг 1: Подключение jQuery.

Для установки необходимо подключить jquery версии 1.7 и выше. Подключение описано ниже.
Внимание! Подключаем только один вариант!

Подключение jquery последней версии

<script src="http://ajax.googleapis.com/ajax/libs/jquery/1/jquery.min.js"></script>

Подключение jquery 1.11.0

<script src="https://ajax.googleapis.com/ajax/libs/jquery/1.11.0/jquery.min.js"></script>

Шаг 2: Подключение ресурсов

Подключаем следующие ресурсы плагина:

Javascript file

<script src="/js/mobileMenu.min.js"></script>

CSS file

<link href="/css/mobileMenu.css" rel="stylesheet" />

Шаг 3: Создание Html-структуры меню

Меню должно состоять из вложенных друг в друга списков, составленных через элементы "<ul></ul>, <li></li>, <a href=""></a>". Пример правильной структуры:

<nav class="menu">
<ul> <li><a href="#">Одежда</a> <ul> <li><a href="#">Кофточки</a><li> <ul> <li><a href="#">Цветные</a></li> <li><a href="#">Полосатые</a></li> </ul> </li> <li><a href="#">Платья</a> <ul> <li><a href="#">Летние</a></li> <li><a href="#">Выпускные</a></li> <li><a href="#">Свадебные</a></li> </ul> </li> <li><a href="#">Современные</a></li> </ul> </li> <li><a href="#">Обувь</a></li> <li><a href="#">Зонты</a></li> <li><a href="#">Шарфы и перчатки</a></li> </ul>
</nav>

Шаг 4: Создание Html-структуры кнопки "Гамбургер"

Для корректной работы меню необходимо создать сруктуру кнопки "Гамбургер" следующего вида:

<span class="toggle js-toggle"> <span> <em></em> <em></em> <em></em> </span> </span>

Шаг 5: Вызов плагина

Вызов плагина осуществляется при помощи применения метода ".mobileMenu()" на обертку общего списка. Ниже представлен пример вызова плагина.

$( document ).ready(function() { $('.menu').mobileMenu(); });

Шаг 6: Скрытие основного меню

Для исправления "эффекта появления меню при загрузке" необходимо в сss файл прописать скрытие меню на ширине, где меню приобретает мобильный вид.

@media(max-width: 899px) { .menu { display: none } }


Параметры плагина

width

Значение ширины экрана в пикселях, меньше которого меню приобретет мобильный вид

Значение по умолчанию: 1024 Возможные значения: Целые числа

next

Html-код кнопки показа следующего уровня меню

Значение по умолчанию: <span class="arrow">›</span> Возможные значения: Любой html-селектор

prev

Html-код стрелки показа предыдущего уровня меню

Значение по умолчанию: <span class="back-arrow">›</span> Возможные значения: Любой html-селектор


Пример вызова плагина со своими параметрами

$( document ).ready(function() { $('.menu').mobileMenu({ 'width': 900, 'next': '<div class="next-arrow next-button">›</div>', 'prev': '<div class="back-arrow back-button">‹</div>', }); });
