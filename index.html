Обзор

Описание

Основные возможности меню

  1. Поддержка бесконечного числа уровней вложенности
  2. Поддержка swipe-раскрытия и swipe-закрытия меню
  3. Адаптивность стилей
  4. Легкая и быстрая настройка

Системные требования

  • Jquery 1.7+
  • Android: 4.2.2+
  • IOS 7.1+

Работа с плагином

Подключение материалов и создание Html-структуры

Шаг 1: Подключение jQuery.

Для установки необходимо подключить jquery версии 1.7 и выше. Подключение описано ниже. 

/*Подключение jquery последней версии*/
<script src="http://ajax.googleapis.com/ajax/libs/jquery/1/jquery.min.js"></script>

Шаг 2: Подключение ресурсов плагина

/*CSS file*/
<link href="/css/mobileMenu.css" rel="stylesheet" />

/*Javascript file*/
<script src="/js/mobileMenu.min.js"></script>

Шаг 3: Создание Html-структуры меню

Меню должно состоять из вложенных друг в друга списков, составленных через элементы: ul, li, a

/*Пример правильной структуры*/
<nav class="menu">
  <ul>
    <li><a href="#">Раздел 1</a></li>
    <li>
      <a href="#">Раздел 2</a>
      <ul>
        <li><a href="#">Подраздел 1</a></li>
        <li>
          <a href="#">Подраздел 2</a>
          <ul>
            <li><a href="#">Продукт 1</a></li>
            <li><a href="#">Продукт 2</a></li>
          </ul>
        </li>
        <li><a href="#">Подраздел 3</a></li>
        <li><a href="#">Подраздел 4</a></li>
      </ul>
    </li>
    <li><a href="#">Раздел 3</a></li>
    <li><a href="#">Раздел 4</a></li>
  </ul>
</nav>

Шаг 4: Создание Html-структуры кнопки "Гамбургер"

Для корректной работы меню необходимо создать сруктуру кнопки "Гамбургер" в любое место страницы:

<span class="multimenuToggle"></span>

Шаг 5: Вызов плагина

Вызов плагина осуществляется при помощи применения метода ".mobileMenu()" на обертку общего списка. Ниже представлен пример вызова плагина.

$( document ).ready(function() {
  $('.menu').mobileMenu();
});

Шаг 6: Скрытие основного меню

Для исправления "эффекта появления меню при загрузке" необходимо в сss файл прописать скрытие меню на ширине, где меню приобретает мобильный вид.

@media(max-width: 899px) {
  .menu {display: none;}
}

Параметры плагина

width

Значение ширины экрана в пикселях, меньше которого меню приобретет мобильный вид

Значение по умолчанию: 1024
Возможные значения: Целые числа

next

Html-код кнопки показа следующего уровня меню

Значение по умолчанию: <span class="arrow"></span>
Возможные значения: Любой html-селектор

prev

Html-код стрелки показа предыдущего уровня меню

Значение по умолчанию: <span class="back-arrow"></span>
Возможные значения: Любой html-селектор

linkSwitch

Разрешает осуществлять навигацию в меню по самой ссылке, а не только по стрелке

Значение по умолчанию: false
Возможные значения: true, false

nextButonTheme

Цвет кнопок плагина

Значение по умолчанию: не задан
Возможные значения: любые цвета в HEX, rgb, rgba

nextButonTheme

Тема для кнопки перехода на следующий уровень

Значение по умолчанию: 'angle-right'
Возможные значения: 'right', 'right-open', 'right-big', 'right-hand', 'right-circled', 'angle-circled-right', 'angle-double-right'

prevButonTheme

Тема для кнопки перехода на предыдущий уровень

Значение по умолчанию: 'angle-left'
Возможные значения: 'left', 'left-open', 'left-big', 'left-hand', 'left-circled', 'angle-circled-left', 'angle-double-left'

openButonTheme

Тема для кнопки раскрытия меню

Значение по умолчанию: 'menu'
Возможные значения: 'th-large', 'th', 'th-list', 'th-1', 'th-list-1'


Пример вызова плагина с собственными параметрами

$( document ).ready(function() {
  $('.menu').mobileMenu({
    'width': 900,
    'next': '<div class="next-arrow next-button"></div>',
    'prev': '<div class="back-arrow back-button"></div>',
    'linkSwitch': true,
    'nextButonTheme' : 'right',
    'prevButonTheme' : 'left',
    'openButonTheme' : 'th-list',
    'colorTheme' : '#f09e00'
  });
});
